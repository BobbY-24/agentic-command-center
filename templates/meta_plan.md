# Meta-Plan

## Restated idea

The idea rewritten as an operational project or decision.

## Decision question

The primary decision Bob needs to make.

## Opportunity thesis

Why the idea may be worth pursuing and what must be true.

## Strategic options

Three to five possible paths, including a park/kill option when useful.

## Worker routing

Which tools or humans should do which work.

## Tool prompt packets

Copy-paste prompts for each worker.

## Deal teams to run

The competing analyst-banker teams needed for self-correction.

## Human data request

Private, observational, preference, or high-stakes data Bob must supply.

## GitHub artifact plan

Files, run folder, and reusable artifacts to save.

## Codex directive

Implementation-ready instructions if execution is useful now.

## LinkedIn/public signal plan

Whether the output should become public, and under what conditions.

## Next 3 actions

The immediate ordered actions Bob should take.
